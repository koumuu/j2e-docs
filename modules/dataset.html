dataset module¶

言語、データセットに関する。

class dataset.Lang¶

ベースクラス: object

言語集。単語を番号つけて、番号と単語の相互な対応はされる。

add_sentence(sentence)¶

文を追加する。

パラメータ

sentence -- 追加する文。日本語に対しては、分かち書きされた文が必要。

戻り値

なし。

add_word(word)¶

単語を追加する。

パラメータ

word -- 追加する単語。

戻り値

なし。

class dataset.ReservedToken¶

ベースクラス: enum.IntEnum

内部保留される単語

EOS = 1¶

End of Sequenceの略。文の終了を表す。

PAD = 2¶

文を固定な長さを保つために、足りない文を補足する。

RESERVED_1 = 4¶

システムの拡張のために保留。

RESERVED_2 = 5¶

システムの拡張のために保留。

RESERVED_3 = 6¶

システムの拡張のために保留。

RESERVED_4 = 7¶

システムの拡張のために保留。

RESERVED_5 = 8¶

システムの拡張のために保留。

RESERVED_6 = 9¶

システムの拡張のために保留。

RESERVED_7 = 10¶

システムの拡張のために保留。

SOS = 0¶

Start of Sequenceの略。文の開始を表す。

UNK = 3¶

データセットが認識できない単語を表す。

static indexes_from_sentence(lang, sentence, max_length, is_source_lang=True)¶

戻り値

class dataset.TranslationDataset(source_lang_file, target_lang_file, max_length, transform=None)¶

ベースクラス: torch.utils.data.dataset.Dataset

パラメータ
