embedding module¶

二つのembeddingネットワークを提供する。

embeddingとは、単語番号をベクトルに変換するネットワーク。

class embedding.FasttextEmbedding(model_file, lang, hidden_size)¶

ベースクラス: object

fasttextを用いたembeddingネットワーク。

パラメータ

  • model_file -- fasttextで訓練済みのモデルのファイル位置。

  • lang -- 対応する言語集のインスタンス。

  • hidden_size -- 単語ベクトルの次元数。

property n_words¶
戻り値

単語総数。

class embedding.NaiveEmbedding(n_words, hidden_size)¶

ベースクラス: torch.nn.modules.sparse.Embedding

torchのembeddingネットワーク。

パラメータ

  • n_words -- 単語総数。

  • hidden_size -- 単語ベクトルの次元数。

property n_words¶
戻り値

単語総数。
